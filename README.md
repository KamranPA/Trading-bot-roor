# 🤖 سیستم معاملاتی هوشمند چند-مدلی (v8.1)

این پروژه یک سیستم معاملاتی الگوریتمی هوشمند است که با بهره‌گیری از **یادگیری ماشین (Machine Learning)**، برای جفت‌ارزهای مختلف به صورت اختصاصی آموزش می‌بیند و به طور خودکار بهینه‌سازی می‌شود.

## 🚀 ویژگی‌های کلیدی
* **معماری Multi-Model:** برای هر ارز یک مغز اختصاصی (`.pkl`) ساخته می‌شود که رفتارِ قیمتیِ همان ارز را به صورت انحصاری یاد می‌گیرد.
* **آموزش پویا (Continuous Learning):** سیستم در پایان هر ماه با ترکیب تجربیات لایو (سود/ضررها) و دیتای تاریخی، خود را بازآموزی می‌کند تا همیشه با شرایط جدید بازار منطبق باشد.
* **مانیتورینگ و Heartbeat:** ارسال گزارش وضعیت سلامت سیستم (هر شب ساعت ۲۲:۰۰) و نوتیفیکیشن‌های ارتقای هوش مصنوعی در تلگرام.
* **مدیریت ریسک هوشمند:** محاسبه خودکار نسبت ریسک به پاداش (R:R) و نمایش جزئیات دقیق در سیگنال‌های ارسالی.
* **پایداری در شبکه:** استفاده از `Proxy` و `Session` جهت عملکرد بی‌نقص در محدودیت‌های شبکه.



## 🛠 ساختار پروژه
```text
├── src/
│   ├── models/           # مغز متفکر (مدل‌های هوش مصنوعی اختصاصی هر ارز)
│   ├── brain.py          # لایه مدیریت و توزیع مدل‌های هوش مصنوعی
│   ├── database.py       # مدیریت دیتابیس (SQL) و ثبت پوزیشن‌ها
│   ├── telegram_bot.py   # لایه ارتباطی و نوتیفیکیشن‌های تلگرام
│   └── train_model.py    # اسکریپت بازآموزی ماهانه و ترکیب دیتا
├── data/                 # محل ذخیره دیتابیس‌های بکتست و لایو
├── main.py               # هسته اصلی و اجرای موازی ربات
└── best_params.json      # پارامترهای بهینه‌شده داینامیک
