# 🤖 ربات تریدر هوشمند کوین‌اکس (CoinEx Trading Bot)

این پروژه یک ربات معامله‌گر خودکار و هوشمند برای صرافی CoinEx است که بر پایه زبان پایتون توسعه یافته است. ربات با ترکیب استراتژی‌های تکنیکال (پرایس اکشن و اندیکاتورها) و فیلترهای پیشرفته هوش مصنوعی (Machine Learning)، سیگنال‌های معاملاتی را شناسایی، مدیریت و اجرا می‌کند.

پروژه به یک سیستم **بکتست، بهینه‌سازی پارامترها و آموزش مداوم هوش مصنوعی خودکار** از طریق GitHub Actions مجهز است.

---

## 📈 واچ‌لیست فعال ربات
ربات در حال حاضر ۱۱ ارز برتر و پرحجم زیر را به صورت همزمان تحت نظر دارد و ترید می‌کند:
* `BTC/USDT` | `ETH/USDT` | `SOL/USDT` | `POL/USDT` | `ADA/USDT`
* `DOT/USDT` | `LINK/USDT` | `UNI/USDT` | `AVAX/USDT` | `XRP/USDT` | `LTC/USDT`

---

## 🛠️ ساختار و معماری پروژه

```text
├── .github/
│   └── workflows/
│       └── run_backtester.yml   # ورک‌فلو خودکار بکتست، آپتیمایزر و ماشین لرنینگ
├── data/
│   ├── 4h/                      # محل ذخیره موقت فایل‌های داده تاریخی CSV
│   └── trading_bot.db          # دیتابیس اصلی معاملات و دیتای ایزوله بکتست
├── src/
│   ├── models/
│   │   └── trading_filter_model.pkl  # مدل هوش مصنوعی آموزش‌دیده (RandomForest)
│   ├── backtester.py            # اسکریپت شبیه‌ساز معاملات گذشته روی ۱۱ ارز
│   ├── optimizer.py             # بهینه‌ساز هوشمند پارامترهای استراتژی (Self-Optimization)
│   ├── train_model.py           # لایه آموزش مدل هوش مصنوعی از روی تریدهای دیتابیس
│   ├── indicators.py            # محاسبات اندیکاتورها (ADX, RSI, ATR, Swing High/Low)
│   └── strategy_utils.py        # ابزارهای کمکی مدیریت استراتژی
├── config.py                    # تنظیمات ساختاری، مسیرهای مطلق دیتابیس و مدیریت واچ‌لیست
├── fetcher.py                   # دریافت‌کننده خودکار داده‌های تاریخی بازار از کوین‌اکس
├── main.py                      # هسته اصلی و لایو ربات تریدر
└── best_params.json             # پارامترهای بهینه‌شده داینامیک (ADX Threshold, TP/SL Ratios)
