# Trading-bot-root 📈

سیستم سیگنال‌دهی هوشمند و خوداصلاح‌گر کریپتو در تایم‌فریم ۴ ساعته بر پایه شکست سطوح سوئینگ، فیلتر حجم و روند (ADX).

## 🛠️ ابزارهای استفاده شده
* زبان برنامه‌نویسی: Python 3.10
* کتابخانه صرافی: CCXT (CoinEx API)
* دیتابیس: SQLite
* بستر اجرا: GitHub Actions (۱۰۰٪ رایگان)

## ⏱️ نحوه کارکرد
این ربات هر ۴ ساعت یک‌بار به صورت خودکار توسط گیت‌هاب اجرا شده، بازار را اسکن می‌کند و سیگنال‌ها را به تلگرام می‌فرستد. اول هر ماه نیز سیستم بهینه‌سازی ماهانه را اجرا می‌کند.
